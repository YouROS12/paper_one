Abstract

We present an intelligent preprocessing framework for plant disease instance segmentation on PlantSeg — the largest in-the-wild plant disease segmentation dataset, comprising 11,458 images across 115 disease categories. Using a single A100 GPU with approximately 9 hours of training per strategy, our bounding-box-guided preprocessing achieves 34.12% mIoU, surpassing DeepLabv3+ ResNet-101 (27.18%) and approaching SAN ViT-B/16 (34.79%) — while requiring 3× fewer parameters, no multi-GPU infrastructure, and remaining fully deployable on edge devices. We further demonstrate that context-aware mask augmentation improves early convergence but degrades final performance, revealing that the perceptual gradient between healthy and necrotic tissue constitutes a diagnostically irreplaceable feature signal. Finally, we show empirically that instance-level metrics (mAP50) expose failure modes invisible to pixel-level metrics (mIoU) — a methodological contribution to how plant disease models should be evaluated.

PlantSeg — Benchmark Context

115Disease Classes
11,458In-the-Wild Images
~104Images / Class avg.

Unlike PlantVillage (38 classes, lab conditions) or PlantDoc (27 classes), PlantSeg is captured entirely in the wild — complex backgrounds, variable lighting, occlusions, morphological variation. The 70/10/20 train/val/test split and ~104 images per class make it among the most visually challenging plant disease benchmarks in existence.

Benchmark Gap

All prior PlantSeg work uses semantic segmentation evaluated on mIoU only. This work establishes the first instance segmentation baseline using a YOLO-family model evaluated on both mAP50 and mIoU.

Annotation Types Available

PlantSeg provides both expert-annotated polygon masks and derived bounding box annotations — enabling the annotation-efficiency analysis central to this work.

03 · Key Findings
What the results say
03

Four Key Findings

Finding 1 — Dual Winner

mask_guided leads mAP50 (32.69%), bbox_guided leads mIoU (34.12%). Margin under 0.5% on both. Jointly optimal for different use cases: bbox_guided for pixel-level disease area quantification; mask_guided for instance-level detection.

Finding 2 — The Context Signal

mask_guided_aware underperforms mask_guided despite using identical spatial guidance plus augmentation. The healthy-to-necrotic boundary suppressed by augmentation is diagnostically informative. The model actively exploits this perceptual gradient.

Finding 3 — The random_crop Anomaly

random_crop ranks last on mAP50 (27.26%) but 5th on mIoU (30.22%) — above baseline. Clearest empirical proof that mIoU and mAP50 measure fundamentally different model capabilities.

Finding 4 — Annotation Efficiency

bbox_guided (bounding boxes only) beats mask_guided on mIoU by +0.40%, losing mAP50 by only −0.44%. Expensive polygon annotation is not justified when pixel-level quantification is the primary goal.

05 · Metrics Analysis
mIoU vs mAP50
05

Why Both Metrics Matter

mIoU — Pixel-level metric

Per-pixel class correctness averaged over classes. Does not penalize missed instances. A class absent from both prediction and GT is skipped — forgiving by design.

mAP50 — Instance-level metric

Requires simultaneous detection + localization (≥50% IoU) + classification of every individual instance. No partial credit. Penalizes every missed detection directly.

The random_crop Proof

5th on mIoU, last on mAP50. Diverse crops improve pixel coverage in aggregate while destroying instance detection. A model appears adequate on pixels while catastrophically failing on instances.

Methodological Contribution

No prior PlantSeg benchmark reports mAP50. This divergence is publishable on its own — it reveals a systematic gap in how plant disease models have been evaluated.

The Core Insight

The healthy-to-necrotic tissue boundary is not background noise — it is a diagnostic feature the network actively exploits. Suppressing it via augmentation carries a measurable, reproducible performance cost. This has not been published on PlantSeg before.
